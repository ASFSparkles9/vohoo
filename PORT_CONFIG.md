# Настройка портов

## Порт бэкенда: 3001

Бэкенд (Next.js) теперь запускается на порту **3001** вместо 3000.

### Запуск бэкенда:

```bash
npm run dev
```

Бэкенд будет доступен на: `http://localhost:3001`

## Настройка мобильного приложения

Обновите файл `.env` в папке `mobile`:

**Для локального тестирования:**
```
EXPO_PUBLIC_API_URL=http://localhost:3001
```

**Для реального устройства:**
```
EXPO_PUBLIC_API_URL=http://192.168.3.3:3001
```
(Замените `192.168.3.3` на IP-адрес вашего компьютера)

## Изменение порта

Если вы хотите использовать другой порт:

1. **Измените порт в `package.json`:**
   ```json
   "dev": "next dev -p ВАШ_ПОРТ"
   ```

2. **Обновите `.env` файлы:**
   - В корне проекта: `NEXT_PUBLIC_APP_URL=http://localhost:ВАШ_ПОРТ`
   - В `mobile/.env`: `EXPO_PUBLIC_API_URL=http://localhost:ВАШ_ПОРТ`

3. **Для реального устройства используйте IP:**
   ```
   EXPO_PUBLIC_API_URL=http://ВАШ_IP:ВАШ_ПОРТ
   ```

## Порт веб-версии Expo

Веб-версия Expo по умолчанию использует порт **19006**. Это можно изменить через переменную окружения `PORT`:

```bash
PORT=8080 npm run web
```
