# Быстрый старт с Ollama

## ⚡ За 2 минуты

### 1. Установка Ollama

**Windows:**
- Скачайте: https://ollama.com/download/windows
- Установите и запустите

**macOS:**
```bash
brew install ollama
```

**Linux:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

### 2. Загрузите модель

```bash
ollama pull llama3.2
```

Это займет несколько минут (модель ~2GB).

### 3. Проверьте

```bash
ollama list
```

Должна появиться модель `llama3.2`.

### 4. Готово! 🎉

Приложение автоматически обнаружит Ollama и начнет использовать его.

## 🧪 Тест

Откройте терминал:
```bash
ollama run llama3.2 "Привет! Как дела?"
```

Если видите ответ - всё работает!

## 📱 В приложении

Откройте раздел "ИИ Помощник" - должен появиться статус "🤖 Ollama активен".

## 🔄 Если не работает

1. Убедитесь, что Ollama запущен (должен быть в трее/процессах)
2. Проверьте: http://localhost:11434
3. Перезапустите приложение

## 💡 Альтернативные модели

```bash
# Легкая модель (быстрее, меньше памяти)
ollama pull phi3

# Более мощная
ollama pull mistral

# Самая мощная (требует много памяти)
ollama pull llama3
```

Затем измените в `lib/ai-local.ts`:
```typescript
model: 'phi3', // или другая модель
```
