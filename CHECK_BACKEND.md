# Проверка работы бэкенда

## ✅ Бэкенд запущен!

Ошибка 405 означает, что сервер работает, но метод запроса неправильный.

## Проверка через браузер:

Откройте в браузере:
```
http://localhost:3001
```

Должна открыться главная страница приложения.

## Проверка API:

API маршрут `/api/init` использует **POST** метод, поэтому в браузере он не откроется напрямую.

### Проверка через PowerShell:

```powershell
Invoke-WebRequest -Uri "http://localhost:3001/api/init" -Method POST -ContentType "application/json"
```

### Проверка через curl (если установлен):

```bash
curl -X POST http://localhost:3001/api/init -H "Content-Type: application/json"
```

## Проверка других API:

Попробуйте открыть в браузере:
```
http://localhost:3001/api/notes
```

Этот маршрут использует GET метод и должен вернуть список заметок (может быть пустым).

## Если бэкенд не отвечает:

1. **Проверьте, что бэкенд запущен:**
   - Должен быть процесс Node.js
   - В терминале должно быть: `Local: http://localhost:3001`

2. **Проверьте порт:**
   ```powershell
   netstat -ano | findstr :3001
   ```

3. **Перезапустите бэкенд:**
   - Нажмите Ctrl+C в терминале бэкенда
   - Запустите снова: `npm run dev`

## Главное:

**Бэкенд должен быть запущен в отдельном терминале!**

Откройте второй терминал и запустите:
```bash
cd F:\vohoo
npm run dev
```
