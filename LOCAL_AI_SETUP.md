# Настройка локальной нейросети

## 🎯 Что это?

Теперь приложение использует **локальную нейросеть**, которая не зависит от внешних API (OpenAI и т.д.). Это означает:
- ✅ Полная приватность данных
- ✅ Работает без интернета (после установки)
- ✅ Бесплатно
- ✅ Нет ограничений по запросам

## 🚀 Варианты установки

### Вариант 1: Ollama (Рекомендуется)

Ollama - это локальный сервер с open-source моделями.

#### Установка:

**Windows:**
1. Скачайте Ollama: https://ollama.com/download
2. Установите и запустите
3. Откройте терминал и выполните:
   ```bash
   ollama pull llama3.2
   ```
   Или другую модель:
   ```bash
   ollama pull mistral
   ollama pull phi3
   ```

**macOS/Linux:**
```bash
curl -fsSL https://ollama.com/install.sh | sh
ollama pull llama3.2
```

#### Проверка:
```bash
ollama list
```

#### Использование:
Приложение автоматически обнаружит Ollama, если он запущен на `http://localhost:11434`.

### Вариант 2: Без Ollama (Простая логика)

Если Ollama не установлен, приложение автоматически использует **простую локальную логику**:
- Анализ недели на основе статистики
- Извлечение задач из заметок
- Простые резюме
- Прогноз энергии на основе данных

## 📝 Как это работает

1. **При запуске** приложение проверяет доступность Ollama
2. **Если Ollama доступен** - использует его для умного анализа
3. **Если Ollama недоступен** - использует простую локальную логику
4. **Все работает** без внешних API!

## 🔧 Настройка модели

По умолчанию используется `llama3.2`. Чтобы изменить модель, отредактируйте `lib/ai-local.ts`:

```typescript
model: 'llama3.2', // Замените на свою модель
```

Доступные модели:
- `llama3.2` - быстрая и эффективная
- `mistral` - хорошая для анализа
- `phi3` - легкая модель
- `llama3` - более мощная (требует больше памяти)

## 📊 Проверка статуса

Откройте в браузере:
```
http://localhost:3001/api/ai/status
```

Ответ:
```json
{
  "ollama": true,
  "method": "ollama"
}
```

или

```json
{
  "ollama": false,
  "method": "local"
}
```

## 🎨 Использование в приложении

Все ИИ-функции теперь работают локально:

1. **Анализ недели** - умный анализ или простая статистика
2. **Создание задач** - извлечение задач из заметок
3. **Резюме заметок** - краткое описание
4. **Прогноз энергии** - предсказание на основе данных
5. **Оптимальное время** - поиск времени для встреч

## ⚡ Производительность

- **С Ollama**: Более умные ответы, но требует больше ресурсов
- **Без Ollama**: Быстрая работа, простая логика

## 🔒 Приватность

- ✅ Все данные обрабатываются локально
- ✅ Ничего не отправляется в интернет
- ✅ Полная конфиденциальность

## 🐛 Устранение неполадок

### Ollama не обнаруживается:

1. Проверьте, что Ollama запущен:
   ```bash
   ollama serve
   ```

2. Проверьте порт:
   ```
   http://localhost:11434
   ```

3. Проверьте модель:
   ```bash
   ollama list
   ```

### Медленная работа:

- Используйте более легкую модель: `phi3` или `llama3.2:1b`
- Или используйте простую локальную логику (отключите Ollama)
