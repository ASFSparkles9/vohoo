# SyncNote — Интеллектуальное приложение для заметок и расписаний

SyncNote — это современное приложение, объединяющее функции личных задач, умного календаря, совместного планирования и ИИ-помощника.

## 🤖 Локальная нейросеть

Приложение использует **локальную нейросеть** (Ollama) или простую локальную логику - **без зависимости от внешних API**!

- ✅ Полная приватность
- ✅ Работает без интернета
- ✅ Бесплатно
- ✅ Нет ограничений

📖 [Инструкция по установке Ollama](LOCAL_AI_SETUP.md) | [Быстрый старт](OLLAMA_QUICK_START.md)

## 🚀 Основные функции

### 📝 Личные заметки
- Теги и папки для организации
- Голосовой ввод
- Прикрепление файлов
- Богатое форматирование

### 📅 Умный календарь
- Просмотр дня, недели и месяца
- Drag & Drop для перемещения событий
- Цветовые категории
- Умное позиционирование событий

### 👥 Стыковка с друзьями
- Общие события
- Перекрытие расписаний
- Поиск подходящего времени для встреч
- Синхронизация календарей

### 🤖 ИИ-функции
- Автоматическое создание задач
- Анализ недели
- Оптимальное время для встреч
- Прогноз энергии
- Подсказки по тайм-менеджменту

### ✨ Дополнительные функции
- Плавающие задачи
- Совместные заметки с ИИ-резюме
- История жизни
- Антипрокрастинационный режим
- Стыковка по геолокации
- Баланс жизненных сфер

## 🛠️ Технологии

- **Frontend**: Next.js 14, React, TypeScript
- **Styling**: Tailwind CSS
- **Database**: Prisma ORM + SQLite
- **AI**: OpenAI API
- **UI Components**: Radix UI

## 📦 Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
npm install
```

3. Настройте переменные окружения:
```bash
cp .env.example .env
# Отредактируйте .env и добавьте ваш OPENAI_API_KEY
```

4. Инициализируйте базу данных:
```bash
npx prisma generate
npx prisma db push
```

5. Запустите приложение:
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.
