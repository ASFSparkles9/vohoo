# 👀 Как посмотреть приложение

## Вариант 1: Веб-версия бэкенда (Рекомендуется)

Это полнофункциональная веб-версия приложения:

1. **Убедитесь, что бэкенд запущен:**
   ```bash
   cd F:\vohoo
   npm run dev
   ```

2. **Откройте в браузере:**
   ```
   http://localhost:3001
   ```

   ✅ Полный функционал
   ✅ Все функции работают
   ✅ Красивый интерфейс

## Вариант 2: Мобильное приложение в браузере

Это мобильная версия, адаптированная для браузера:

1. **Убедитесь, что Metro Bundler запущен:**
   ```bash
   cd F:\vohoo\mobile
   npm start
   ```

2. **В терминале Metro нажмите `w`** или откройте:
   ```
   http://localhost:8081
   ```

   ✅ Мобильный интерфейс
   ✅ Адаптивный дизайн
   ⚠️ Некоторые нативные функции не работают

## Вариант 3: На телефоне через Expo Go

1. Установите Expo Go на телефон
2. Отсканируйте QR-код из терминала Metro
3. Приложение загрузится на телефон

   ✅ Полный мобильный опыт
   ✅ Все функции работают
   ✅ Нативные возможности

## Быстрый запуск всего:

### Терминал 1 - Бэкенд:
```bash
cd F:\vohoo
npm run dev
```
Откройте: http://localhost:3001

### Терминал 2 - Мобильное приложение:
```bash
cd F:\vohoo\mobile
npm start
```
Нажмите `w` для веб-версии или отсканируйте QR для телефона

## Что вы увидите:

### Веб-версия (localhost:3001):
- Главная страница с заметками
- Боковое меню со всеми разделами
- Полный функционал приложения

### Мобильная версия (localhost:8081):
- Табы навигации внизу
- Мобильный интерфейс
- Адаптированный под телефон дизайн

## Горячие клавиши Metro:

- `r` - Перезагрузить приложение
- `w` - Открыть в браузере
- `j` - Открыть отладчик
- `m` - Показать меню
- `?` - Показать все команды

## Отладка:

### Открыть консоль разработчика:
- **Веб-версия:** Нажмите `F12`
- **Мобильная версия:** Нажмите `F12` или `Ctrl+Shift+I`

### Проверить API:
```
http://localhost:3001/api/notes
http://localhost:3001/api/events
http://localhost:3001/api/tasks
```

## Рекомендация:

Для разработки используйте **веб-версию бэкенда** (localhost:3001) - это самый быстрый способ увидеть результат!
